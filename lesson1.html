Hello world

В программировании принято начинать знакомство с Hello world.
HTML (hypertext markup language) — гипертекстовый язык разметки. Но от традиций отступать не будем: 

	<!DOCTYPE html>
	<html>
	<head>
		<meta charset="UTF-8">
		<title>
			Знакомство с HTML </title>
	</head>
	<body>
	<h1>Hello World!</h1>
	<!-- Это комментарий в html -->
	</body>
	</html>
Новые тэги здесь все.
<!DOCTYPE> - дескриптор описания типа документа. В данном примере соответствует html5.
<head> - Шапка документа. Не отображается, содержит техническую информацию.
<title> - Заголовок страницы. Всегда распологается внутри шапки.
<body> - Тело документа. Содержимое страницы размещается здесь.
<h1> - Заголовок. Всего их 6 от 1 до 6. Желательно размещать не более одного заголовка h1 на страницу.
Все теги должны быть закрыты, согласно стандарту xhtml. Есть самозакрывающиеся (img, br...) и требующие закрывающего тэга (div, p, a...)
Тэги могут быть вложены друг в друга, но не могут пересекаться. 
<div><p></div></p>

Тэги регистронезависимые.

CSS. Hello... style?!

CSS - каскадные таблицы стилей. 

	.smile{
	font-size: 200px;
	width: 1em;
	height: 1em;
	margin: 0 auto;
	}

	div.code{
	width: 50%;
	display: inline-block;
	color: yellow; /*Желтый*/
	}

	div.code-description{
	width: 40%;
	display: inline-block;
	margin: 20px;
	vertical-align: top;
	color: #FACE8D; /*Телесный*/
	}

	/*Это комментарий в CSS*/
	code{
	border-radius: 10px;
	margin: 10px;
	font-family: monospace;
	color: rgba(255, 0, 0, 0.5); /*Красный с 50% прозрачности*/
	}
Любое правило в CSS начинается с селектора и заключается в фигурные скобки
Численное значение размеров и отступов должно иметь размерность, если оно больше 0: 
(25px, 13%, 1em, 1.5pt)

Цвет задается либо анлийским названием, либо шестнадцатиричным кодом #RRGGBB, либо через компонентный ввод rgba(r,g,b,alpha);
