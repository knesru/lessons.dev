CSS. Селекторы

CSS - каскадные таблицы стилей. 

	/*Селектор элемента*/
	div {
		/*Свойства*/
		width: 100%;/*Ширина 100%*/
		height: 200px;/*Высота 200 пикселей*/
	}

	/*Селектор класса*/
	.someClass {
		color: red;/*Красный цвет текста*/
	}

	/*Селектор id*/
	#foreveralone {
		font-size: 10em;/*Размер шрифта в 10 раз больше, чем у родителя*/
	}

	/*Группировка*/
	h1, h2, h3 {
		border-bottom: 1px solid #eee;/*тонкий серый сплошной бортик внизу*/
	}

	/*Селектор элемента И класса*/
	p .styledParagraph {
		text-indent: 4em;/*отступ равен 4 высотам текста*/
	}

	/*Селектор элемента внутри другого элемента*/
	span.nicelink a{
		background-color: #336699; /*Фон темно-циановый*/
	}
Обратите внимание: #eee - сокращенная запись #eeeeee;
Селектор элемента внутри другого элемента - это одно из проявлений каскадов. Данное правило будет применено только к ссылке, которая находится где-то внутри такого span, который имеет класс nicelink.
N.B. Все названия классов и ID выдуманные. Названия элементов - реальные.

HTML. Классы, id


	<div> Плашка 100% ширины, 200 пикселей в высоту </div>
	<div class="someClass">
		Плашка 100% ширины,
		200 пикселей в высоту
		с красным шрифтом
	</div>
	<i id="foreveralone"> очень крупный шрифт </i>
	<!-- все три заголовка будут иметь подчеркивание -->
	<h1> Заголовок </h1>
	<h2> ПодЗаголовок </h2>
	<h3> Под-ПодЗаголовок </h3>
	<p> Этот абзац без стилей </p>
	<p class="styledParagraph"> Этот абзац будет стилизован </p>
	<a href="http://google.com"> Просто ссылка </a>
	<span> <a href="http://google.com"> Просто ссылка </a></span>
	<div class="nicelink"> <a href="http://google.com"> Просто ссылка </a> </div>
	<span class="nicelink">
		<a href="http://google.com">
			Стилизованная ссылка
		</a>
	</span>
	<span class="nicelink">
		<b>
			<a href="http://google.com"> Стилизованная ссылка </a>
		</b>
	</span>
id и class - аттрибуты. id элемента уникален. Будет большой ошибкой создать несколько элементов с одинаковым id. Если требуется создать несколько элементов, имеющих одинаковое или очень похожее оформление, используйте class

CSS псевдоселекторы

Это такие селекторы, но не совсем©
Данные селекторы производят выборку на основе логических данных, а не физических. 

	/*Селектор всех элементов*/
	*{
		vertical-align: top; /* сверстать всех наверх */
	}

	/*некоторые псевдо-классы*/

	/*Наведение курсора*/
	a:hover{
		outline: 1px solid green;
	}

	/*Курсор нажат*/
	a:active{
		margin: 1px 1px -1px -1px;
	}

	/*Выберет активное поле ввода*/
	input:focus{
		outline: none;
	}

	/*Псевдо-элементы*/

	/*Для размещения контента средствами CSS*/
	strong::after{
		content: url(image.gif);
	}
	h2::before{
		content: '!!!';
	}

	/*Для выбора определенных логических частей*/
	/*Будет выбрана первая буква*/
	span::first-letter{

	}
	/*Будет выбрана первая строка*/
	p::first-line{
	}
	/*Применено к выделению*/
	div::selection{
	}
Content не рекомендуется использовать, так как он нарушает разделение между стилями и версткой
